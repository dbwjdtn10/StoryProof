# 스플래시 이미지 커스터마이징 가이드 (5개 랜덤 이미지)

## 현재 상태
- **5개의 이미지 중 랜덤하게 하나**가 전체 화면에 표시됩니다
- 이미지가 없으면 기본 BookOpen 아이콘이 표시됩니다

## 커스텀 이미지 추가 방법

### 1단계: 이미지 준비
- **형식**: PNG 또는 JPG
- **크기**: 1920x1080px 이상 권장 (16:9 비율)
- **스타일**: 소설/스토리 관련 이미지, 일러스트, 또는 원하는 배경 이미지
- **개수**: 최소 1개 ~ 최대 5개

### 2단계: 파일 이름 지정
이미지 파일의 이름을 다음과 같이 변경:
- `splash1.png`
- `splash2.png`
- `splash3.png`
- `splash4.png`
- `splash5.png`

> **참고**: 5개 모두 필요하지 않습니다. 1개만 있어도 작동하며, 있는 이미지 중에서만 랜덤 선택됩니다.

### 3단계: 파일 배치
준비한 이미지들을 이 폴더(`backend/static/images/`)에 복사

### 4단계: 확인
- 서버 재시작 (또는 브라우저 새로고침)
- 앱 접속 시 랜덤하게 선택된 이미지가 전체 화면에 표시됩니다

## 디자인 팁
- **비율**: 16:9 (가로형) 권장 - 전체 화면에 최적화
- **해상도**: 최소 1920x1080px (Full HD)
- **밝기**: 너무 밝지 않은 이미지 권장 (하단 텍스트 가독성)
- **테마**: 소설, 책, 글쓰기, 스토리텔링 관련 이미지가 어울립니다

## 예시 이미지 아이디어
- 책상 위의 노트북과 커피
- 도서관 서가
- 빈티지 타자기
- 책 더미
- 펜과 종이
- 판타지/로맨스 일러스트
- AI 생성 소설 관련 이미지

## 화면 구성
- **이미지**: 전체 화면 배경 (어두운 오버레이 적용)
- **브랜드명**: 하단 중앙
- **프로그레스 바**: 브랜드명 아래 (6시 방향)
- **로딩 메시지**: 프로그레스 바 아래
