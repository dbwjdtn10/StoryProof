# 개인정보 처리방침 (초안)

> **주의: 이 문서는 법률 자문을 대체하지 않는 초안입니다.** 실제 서비스에
> 게시하기 전 반드시 개인정보보호 전문 변호사/노무사의 검토를 받으세요.
> `[ ]`로 표시된 항목은 사업자가 직접 채워야 하는 미확정 값입니다.
> 작성 기준: 개인정보 보호법(PIPA) 및 표준지침. 최종 업데이트: 2026-07-13.

StoryProof(이하 "회사")는 이용자의 개인정보를 중요시하며, 「개인정보 보호법」을
준수합니다. 본 처리방침을 통해 이용자가 제공하는 개인정보가 어떠한 용도와
방식으로 이용되고 있으며, 개인정보보호를 위해 어떠한 조치가 취해지고 있는지
알려드립니다.

## 1. 수집하는 개인정보 항목 및 수집 방법

### 1-1. 자체 서비스(B2C) 이용자
| 구분 | 항목 | 수집 방법 |
|---|---|---|
| 필수(회원가입) | 이메일, 사용자명(username), 비밀번호(해시 저장) | 회원가입 시 직접 입력 |
| 자동 수집 | 접속 IP(레이트 리밋 목적, 로그 미영구저장), 최종 로그인 일시 | 서비스 이용 과정에서 자동 생성 |
| 이용자 생성 콘텐츠 | 업로드한 소설 원고, 챕터 본문, AI 챗봇 대화 이력 | 이용자가 직접 업로드/입력 |

비밀번호는 원문이 아닌 단방향 해시(bcrypt/argon2)로 저장되며 회사도 원문을
알 수 없습니다.

### 1-2. B2B 파트너사 API 이용
파트너사가 API로 전송하는 원고 데이터는 파트너사와의 별도 계약(데이터 처리
위탁 계약)에 따라 처리되며, 파트너사가 자사 최종 이용자(독자)의 개인정보를
직접 수집하는 경우 그 처리는 파트너사의 개인정보 처리방침을 따릅니다.
회사는 파트너사의 **수탁자(processor)**로서 API 요청에 포함된 텍스트
데이터만을 계약된 목적(Q&A/일관성 검증 등)으로 처리합니다.

## 2. 개인정보의 수집 및 이용 목적

- 회원 가입의사 확인, 본인 식별·인증, 회원자격 유지·관리
- 소설 분석/Q&A/캐릭터 챗봇 등 서비스 제공
- 부정 이용 방지 및 서비스 남용 차단(레이트 리밋)
- B2B 파트너 대상 사용량 정산 및 계약 이행

## 3. 개인정보의 보유 및 이용 기간

- 원칙적으로 회원 탈퇴 시 지체 없이 파기합니다(계정 삭제 API `DELETE /api/v1/auth/me`).
- 단, 관계 법령에 따라 보존이 필요한 경우 해당 법령이 정한 기간 동안 보관합니다:
  - 전자상거래법 등에 따른 계약/청약철회 기록: [ ]년 (해당 시)
  - 통신비밀보호법에 따른 로그인 기록: 3개월
- **콘텐츠 보안 계약(minimal retention 파트너)**: 원고 원문은 처리 완료 즉시
  삭제되고 벡터 임베딩과 청크 단위 메타데이터만 보관됩니다(`docs/PARTNER_API.md`
  §콘텐츠 보안 계약 참고).

## 4. 개인정보의 제3자 제공

회사는 이용자의 동의 없이 개인정보를 외부에 제공하지 않습니다. 다만
아래의 경우는 예외로 합니다:
- 이용자가 사전에 동의한 경우
- 법령의 규정에 의거하거나 수사 목적으로 관계기관의 적법한 절차에 따른 요청이 있는 경우

## 5. 개인정보 처리 위탁 (제3자 처리업체)

서비스 제공을 위해 아래 업체에 데이터 처리를 위탁하고 있습니다:

| 수탁업체 | 위탁 업무 | 처리 데이터 | 처리 위치 |
|---|---|---|---|
| Google (Gemini/Imagen API) | AI 분석·Q&A·이미지 생성 | 업로드 원고 텍스트, 질의 내용 | Google 클라우드 인프라(리전은 Google 정책에 따름) |
| Pinecone | 벡터 검색 인덱스 저장 | 원고 텍스트 임베딩(벡터) 및 청크 텍스트 | Pinecone 클라우드 인프라 |
| [클라우드/호스팅 사업자] | 서버 호스팅 | 전체 서비스 데이터 | [ ] |

각 수탁업체와는 개인정보 처리 위탁 계약을 체결하고 있으며, 위탁업무 내용이나
수탁자가 변경될 경우 본 방침을 통해 공지합니다.

## 6. 이용자의 권리와 행사 방법

이용자는 언제든지 다음의 권리를 행사할 수 있습니다:
- 개인정보 열람·정정·삭제 요구
- 처리 정지 요구
- 회원 탈퇴(계정 삭제): 마이페이지 내 계정 삭제 기능 또는 아래 문의처로 요청

## 7. 개인정보의 안전성 확보 조치

- 비밀번호 단방향 암호화 저장(bcrypt/argon2)
- 파트너 API 키 SHA-256 해시 저장(원본 키는 발급 시 1회만 노출)
- 웹훅 페이로드 HMAC-SHA256 서명으로 위변조 방지
- 소유권 기반 접근 제어(테넌트 격리) — 파트너/이용자는 자신의 데이터만 조회 가능
- 관리자 전용 기능에 대한 별도 권한 검증(`require_admin`)
- 프로덕션 환경 SECRET_KEY 기본값 사용 차단(가드 로직)
